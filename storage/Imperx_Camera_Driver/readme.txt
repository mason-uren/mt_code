To install the camera driver follow the steps outlined bellow

1) Unzip the file provided to us by Imperx
2) Configure the shell script to be executable using chmod +x
3) Run the shell script
4) Follow the onscreen insruction and specify where you would like it to install the driver and sk
5) In the folder IpxCameraSDK_1.2.0.13 there is a bin folder. Cd into this folde6) Run the script ./IpxPlayer
7) A gui should appear
8) Press connect and select the camera interface (usb 3)
9) Press run and a image should appear
